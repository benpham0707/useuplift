## üéØ Analysis Plan for 120 Essays

Once we have 120 essays:

### **Phase 1: Pattern Identification by Prompt**
For each prompt, identify:
1. Common structural patterns (opening ‚Üí body ‚Üí closing)
2. Successful topic choices within prompt
3. Average dimension scores for Tier 1 vs Tier 2
4. Required vs optional elements
5. Red flags that led to rejection (if available)

### **Phase 2: Cross-Prompt Pattern Analysis**
1. Universal patterns across all prompts (vulnerability, specificity, etc.)
2. Prompt-specific patterns (leadership needs quantified impact, creative needs process)
3. Tier 1 differentiators (what separates Harvard from UC Berkeley?)
4. Dimension correlation (which dimensions predict overall success?)

### **Phase 3: Rubric Calibration**
1. Score all 120 essays with current rubric
2. Compare scores to actual admit outcomes
3. Adjust dimension weights if needed
4. Validate tier boundaries (90+ = Tier 1, etc.)
5. Update benchmarks per prompt

### **Phase 4: Training Data Integration**
1. Add exemplar patterns to prompt intelligence
2. Update adaptive rubric with prompt-specific insights
3. Enhance improvement roadmap with exemplar-based suggestions
4. Add "compare to exemplar" feature
